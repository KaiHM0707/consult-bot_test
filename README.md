# Business Consulting AI Bot

An AI-powered web application that acts as a business consultant for students and early-stage founders.
The system will analyze business ideas and returns structured feedback on value proposition, risks, and strategic next steps.

## Tech Stack
- Python
- Flask
- OpenAI API

## Test version feature
- Successfully analyze the following prompt: "Analyze and make a guide for a coffee shop startup.
- Prompts are limited to the prompt mentioned on the previous line
- Features available with flasks and others are currently unavailable
- The features are accessible via Terminal

## Prototype Features (v1)
- Business idea analysis
- Structured consultant-style responses
- Simple web-based chat interface

## Out of Scope (v1)
- User accounts and authentication
- Real-world market data
- Automated financial systems
- Diagram or chart generation

## Planned Extensions
- Business framework modes (SWOT, Lean Canvas, Customer Persona)
- Visual report and diagram generation
- Session memory and exportable summaries

Early development.
